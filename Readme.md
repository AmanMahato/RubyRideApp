To run this project, we need the following:
1. Jave 7 or higher
2. Any IDE (I have used Intellij Idea)
3. Maven 3.0 or higher
4. MySql database
5. Neo4j configured to run at local host running at local host at port 7474 with password configured as "root"
6. Tomcat 9.0 or higher

Steps to run the project:
1. First load the project in any IDE of your choice as a Maven Project
2. Configure the tomcat server
3. Configure the Mysql server and Neo4j databases in your local machine. For mySql, the mysql script is provided with the project zip.
4. Deploy the project on local tomcat server and the maven will build it and deploy as web app.
